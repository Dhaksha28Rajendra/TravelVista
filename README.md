# 🌍 TravelVista - Responsive Travel Agency Website

A modern responsive travel agency landing page developed as part of the **Codveda Technologies Web Development Internship**.

## 📌 Task

**Level 2 – Task 1: Responsive Web Design with Flexbox and Grid**

This project demonstrates:

- Responsive web design
- CSS Flexbox
- CSS Grid
- Mobile-first responsive layout
- Modern UI/UX

---

## ✨ Features

- Responsive Navigation Bar
- Hero Section
- Popular Destinations
- Tour Packages
- Travel Gallery
- Latest Travel Blogs
- Newsletter Subscription
- Professional Footer
- Mobile Navigation
- Smooth Scrolling

## 🛠 Technologies Used

## ▶️ How to Run

1. Open the project folder in Visual Studio Code.
2. Install the **Live Server** extension.
3. Open **index.html**.
4. Click **Go Live**.
